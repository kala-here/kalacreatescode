BOOTCAMPR

An example of a dynamic (users can manipulate content) website, BOOTCAMPR allows users to create a profile, post events, post projects, and show interest in both.

Users can sign in with GitHub & the BOOTCAMPR twitter account automatically tweets about new projects and events. A weekly automated newsletter is sent out with the latest projects and events as well.

BOOTCAMPR was completed in 8 days with a team of 4. We practiced test-driven-development with an almost 100% Simplecov rating.

Stack: Rails, PostgreSQL
